# Global Education Hub

A modern, responsive educational portal built with React, TypeScript, and Tailwind CSS. Features Apple-level design quality with full PWA support for mobile installation.

## Features

- 📱 **Progressive Web App (PWA)** - Installable on mobile devices
- 🎨 **Apple-level Design** - Superior typography and animations
- 🌐 **Bilingual Support** - English and Assamese
- 📖 **Reader Mode** - E-ink optimized reading experience
- 📰 **Latest Education News** - Filtered by region (Assam, India, Global)
- 🏆 **Scholarship Tracking** - Deadlines and eligibility information
- 📅 **Exam Tracker** - Countdown and reminder system
- 🏫 **Top Institutes** - Global university information
- 📱 **Fully Responsive** - Optimized for all screen sizes
- ⚡ **Lightning Fast** - Vite-powered build system

## Tech Stack

- **Frontend Framework:** React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4.1
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Build Tool:** Vite 6.2
- **Deployment:** Vercel

## Quick Start

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Deployment to Vercel

1. Push to GitHub
2. Import to Vercel
3. Deploy!

Or use Vercel CLI:
```bash
vercel
```
